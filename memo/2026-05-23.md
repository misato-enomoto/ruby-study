# Javaとの違い
　ガベージコレクションはrubyもjavaもあるがjavaの方がより高度
　rubyでガベレージコレクション系で気を付けること
　　1. 大量データを一括でメモリに載せない
　　2. ループ内で不要な文字列・配列・Hashを作らない
　　3. ActiveRecordオブジェクトを作りすぎない
　　4. N+1で不要なオブジェクト生成を増やさない
　　5. バッチ処理では find_each / in_batches を使う
　　6. pluck / select を使い、必要なカラムだけ取る
　　7. メモリ使用量をログ・監視で確認する
